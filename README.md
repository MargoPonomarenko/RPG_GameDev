# RPG_GameDev
Implementation of a RPG game developed using Unity

### Суть гри
Пройти основний сюжетний квест, здалати ворогів та отримати артефакти в нагороду. У процесі гри також є можливість досліджувати додаткові локації.

---

### Правила гри
1. **Дослідження світу**:
   - Гравець може досліджувати відкритий світ, взаємодіяти з NPC, знаходити корисні предмети та проходити квести.
   - Деякі локації доступні лише після виконання певних завдань.
2. **Бойова система**:
   - Бій відбувається в покроковому режимі. Гравець обирає дії: звичайна атака, магічна атака (використовує магічні бали) та спроба уникнення бою.
   - Перемога над ворогами приносить цінні нагороди: досвід, зілля, зброю, обладунки та інші корисні предмети.
3. **Розвиток персонажа**:
   - Під час гри персонаж отримує досвід, підвищує рівень і покращує свої характеристики (сила, захист тощо).
